## Visualización de Datos

En esta semana se trabajó la visualización de datos para analizar información, identificar patrones y comprender relaciones entre variables.

## Contenido

- Histogramas, boxplot, scatter plot y gráficos de barras  
- Uso de Matplotlib y Seaborn  
- Pairplot, heatmap y violin plot  
- Ejercicio integrador con dataset de clientes  

## Objetivo

Aplicar distintos tipos de gráficos según el análisis para interpretar datos y obtener conclusiones.
