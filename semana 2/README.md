# Descripción

Este repositorio contiene las actividades realizadas durante la segunda semana de la materia Minería de Datos. Se incluyen ejercicios prácticos enfocados en el análisis, procesamiento y exploración de datos.

## Contenidos

- Tipos de datos y su clasificación  
- Limpieza de datos (valores faltantes e inconsistencias)  
- Transformación de variables categóricas  
- Análisis exploratorio de datos (EDA)  
- Estadísticas descriptivas y correlación  
- Detección de outliers mediante IQR
